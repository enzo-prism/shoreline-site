## Current brand implications

The brand should not pretend to be a mature media network yet. The audience is
small, the channel is early, and the biggest unlock is discoverability.

The website should prioritize:

- Searchable episode titles.
- Topic pages for recurring ideas.
- Timestamped notes and chapters.
- Resources mentioned in each episode.
- Tool pages for products discussed repeatedly.
- A clear channel/show description, since the YouTube channel description is
  currently empty.

## YouTube podcast guidance to respect

YouTube's own podcast discovery guidance says a podcast show on YouTube is a
playlist, and episodes are videos in that playlist. It also recommends using
the same podcast name as the podcast title, keeping full-length episodes in one
public podcast show, and adding a detailed description to improve discovery.

Source: https://support.google.com/youtube/answer/12950577?hl=en

For Shoreline, this means:

- Use `Shoreline` as the show title.
- Keep `Podcast` out of the title unless required by the platform.
- Build a strong show description now.
- Keep clips, shorts, and resource content separate from the full episode
  podcast playlist if those formats are added later.
